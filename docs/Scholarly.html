1. Introduction

1.1 Purpose

The purpose of this project is to develop a flexible web-based data visualization tool for Screen Actors Guild (SAG) Awards actor nominations. The tool will utilize a custom-built REST/GraphQL API to access data related to actor nominations, which will be sourced from The Movie Database (TMDb). The system will provide support for generating statistics and visualizations using at least three different methods, which can be exported in CSV, WebP, and SVG formats. Additionally, the website will support the display of news articles related to each nominated actor, based on configurable external data sources. The goal is to provide users with an intuitive and user-friendly interface for exploring and analyzing data related to actor nominations at the SAG Awards.

1.2 Intended audience and Reading Suggestions

This document was written in order to help people understand the purpose of this website, how it was made, what functionalities it has and what services it provides. The document should be read in the order it is written.

1.3 Produc scope

The Actvis project is a web-based data visualization tool designed for both casual and professional users who want to explore and analyze data related to Screen Actors Guild (SAG) Awards actor nominations. The goal is to create an intuitive and user-friendly platform for exploring and analyzing data related to actor nominations at the SAG Awards.

2. Overall Description

2.1 Product Perspective

This is a new, self contained product. It is a software -> website which will provide InfoGraphics to all the users that need them, in addition to news and other information agout the cinema area. The two components of the product are the front-end, the user interface and the back-end, the web service that serves the users.

2.2 Product Functions

  • Navigation without authentification
  • Search by actor names, movie names, TV shows names and certain categories of cinema (drama, cartoon, action, love story, mistery, fiction, anime) or awards category (best actor in major role, best supporting actor, best soundtrack, best director etc.)
  • Get news about the current editionm of the SAGAwars in the Actors and Latest News pages
  • Get the latest news about all the cinema area, not just the SAGAwars
  • Get recommandations of upcomming movies and series
  • Navigate with the dropdown navbar
  • Get InfoGraphics about all the SAGAwards editions starting from 1994

2.3 Operating Environment

The website will operate in Mozilla Firefox, Microsoft Edge and Google Chrome browsers for the desktop users, using windows 11, windows 10, windows 8.1, windows 7, Linux Or Mac and for tablets such as Ipad, Ipad Pro, Galaxy Note 9, and also for mobile devices such as Samsung Galaxy S9, Iphone 6/7/8/SE/XR, Microsoft Lumia 950, Nexus 6P Android and so on.

3. Page Functionality and Usage

In this section, we cover exactly what each page contains, what it allows the user to do and how it can be used.

3.1 The Home Page

The Home page is the first thing the user sees when they visit our website. It serves as an introduction to the different sections of the website and provides a sneak peek of what the user can expect to find. The header of the page contains a navigation bar, which allows the user to easily access other parts of the website. Additionally, it provides a search option, making it easier for the user to find the information they need.

The Latest news section is prominently featured on the Home page, displaying top news stories in chronological order. Users can navigate to the news page to read more by clicking on the "See More" button. The Interviews section contains a collection of video interviews with actors who are nominated for the current SAG Awards. These interviews provide a unique insight into the lives and experiences of the actors.

The Recommandation section presents the user with an array of upcoming movie trailers. These trailers are carefully selected to provide users with a glimpse of the latest and greatest movies that are soon to hit the big screen. This section is perfect for movie enthusiasts who are looking for their next big blockbuster hit.

Finally, the footer of the Home page contains links to our social media accounts, including Facebook, Twitter, and Instagram. Users can easily follow us on these platforms to stay up-to-date with the latest news and updates. The footer also contains a link to our terms and policy page, which outlines our policies regarding the use of the website.

3.2 The Latest News Page

The Latest News Page has been designed to keep our users informed about the latest news and updates related to the SAG Awards and the film industry. It is an essential source of information for anyone who wants to stay up-to-date with the latest happenings in the world of entertainment.

Similar to the home page, the Latest News Page features a header that provides users with easy access to other parts of our website. Users can also use the search function to find specific news articles and updates that they may be interested in.

The page is divided into five sections that cover a wide range of topics:

  • The "Who are the winners?" section provides users with information about the winners in different categories at the SAG Awards. It features pictures of the winners and a brief description of their achievements.
  • The "Major News" section is dedicated to the most significant news stories of the week. It covers all the important events that have taken place in the world of entertainment.
  • The "More to Explore" section is a collection of video interviews with the actors nominated for the current SAG Awards. Users can watch these interviews to get a better understanding of the actors and their work.
  • The "Exclusive content" section features articles about the SAG Awards and exclusive information provided by the organizers. It is an excellent source of information for anyone who wants to learn more about the awards and the people behind them.
  • The "Last Week's Hottest News" section provides users with a recap of the most significant news stories from the previous week. It is an excellent way to catch up on any news that you may have missed.

This page contains the same footer as the home page.
